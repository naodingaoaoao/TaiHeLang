# 太和编程语言规范 v0.1
## 结合C++性能与Python简洁性的中文编程语言

### 设计理念
- **源于C++**：静态类型、编译到LLVM IR、高性能、面向对象
- **Python般简单**：简洁语法、类型推断、缩进结构、丰富的内置数据结构
- **中文友好**：中文关键字、中文文档、中文错误信息


## 1. 核心语法变革

### 1.1 使用缩进代替花括号
```
函数 主():
    输出("你好，世界！")
    返回 0

如果 x > 0:
    输出("正数")
否则:
    输出("非正数")
```

### 1.2 简化类型声明（类型推断为主）
```
# 类型推断
变量 x = 10          # 自动推断为整数
变量 y = 3.14        # 自动推断为浮点
变量 姓名 = "张三"    # 自动推断为字符串
变量 标志 = 真       # 自动推断为布尔

# 需要时显式声明类型
变量 年龄: 整数 = 18
变量 pi: 浮点 = 3.14159
```

### 1.3 简化关键字列表
```
# 控制流
如果 否则 循环 当 对于 在 中断 继续 返回

# 类型系统
整数 浮点 字符 字符串 布尔 空值 数组 列表 字典 元组

# 面向对象
类 新建 这 继承 重写 接口 抽象 枚举

# 函数和变量
函数 变量 常量 导入 从 作为 导出

# UI相关
窗口 组件 布局 事件 属性 样式

# 值
真 假 空
```

## 2. Python风格的简洁特性

### 2.1 列表推导式
```
# 生成1到10的平方列表
变量 平方列表 = [x * x 对于 x 在 范围(1, 11)]

# 过滤偶数
变量 偶数 = [x 对于 x 在 范围(1, 21) 如果 x % 2 == 0]
```

### 2.2 字典推导式
```
变量 平方字典 = {x: x*x 对于 x 在 范围(1, 6)}
```

### 2.3 切片操作
```
变量 列表 = [1, 2, 3, 4, 5]
变量 前三个 = 列表[0:3]    # [1, 2, 3]
变量 倒数两个 = 列表[-2:]  # [4, 5]
变量 反转 = 列表[::-1]    # [5, 4, 3, 2, 1]
```

### 2.4 解构赋值
```
变量 坐标 = (10, 20)
变量 x, y = 坐标

变量 字典 = {"姓名": "张三", "年龄": 20}
变量 姓名, 年龄 = 字典["姓名"], 字典["年龄"]
```

## 3. C++风格的性能特性

### 3.1 静态类型
```
# 编译时类型检查，避免运行时错误
函数 相加(a: 整数, b: 整数) -> 整数:
    返回 a + b

# 错误：类型不匹配（编译时报错）
相加(10, "字符串")  # 编译错误！
```

### 3.2 值类型和引用类型
```
# 值类型（默认，栈上分配，高性能）
变量 x: 整数 = 10

# 引用类型（使用 & 符号，像C++引用）
变量 y: &整数 = x  # y是x的引用
y = 20            # 修改y也会修改x

# 指针类型（使用 * 符号，像C++指针）
变量 p: *整数 = 新建 整数(10)
```
## 4. 简化的面向对象系统

### 4.1 类定义
```
类 学生:
    变量 姓名: 字符串
    变量 年龄: 整数
    
    # 构造函数（简化）
    函数 __新建__(姓名: 字符串, 年龄: 整数):
        这.姓名 = 姓名
        这.年龄 = 年龄
    
    # 方法（简洁）
    函数 自我介绍() -> 字符串:
        返回 f"我叫{这.姓名}，今年{这.年龄}岁"
    
```
### 4.2 多重继承
```
类 飞行物:
    函数 飞行():
        输出("飞行中...")

类 游泳物:
    函数 游泳():
        输出("游泳中...")

# 多重继承
类 水陆空载具(飞行物, 游泳物):
    函数 运行():
        这.飞行()
        这.游泳()
```

## 7.DLL的编写与调用

### 7.1 Dll文件编写规范
```
DLL函数 add(整数 a, 整数 b):
    返回 a + b
```

### 7.2 Dll文件编译指令
```bash
taihe run xxx.th --dll
```
### 7.3 Dll文件调用规范示例
```
函数 主():
    lib = dll("dll.dll")

    lib.add = ([整数, 整数], 整数)

    result = lib.add(1, 2)

    输出(result)
    
    返回 0
```
